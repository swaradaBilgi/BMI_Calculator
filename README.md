# BMI Calculator
This is a simple BMI (Body Mass Index) calculator built using HTML, and JavaScript. It allows users to input their weight and height to calculate their BMI and determine their weight category.

## Features
- User-friendly interface for inputting weight and height.
- Real-time BMI calculation and display of results.
- Caregory classification based on BMI values (Underweight, Normal weight, Overweight, Obesity).
- Responsive design for use on various devices.
- Delete BMI records stored in the database.

## How to Use
1. Clone the repository or download the source code.
2. Open the `index.html` file in a web browser. 
3. Enter your weight in kilograms and height in centimeters.
4. Click the "Calculate BMI" button to see your BMI and weight category.

## Technologies Used
- **HTML**: For structuring the web page.
- **JavaScript**: For handling the BMI calculation logic and user interactions.
- **Python**: For all backend processing
- **SQLite**: For storing user data and BMI records.

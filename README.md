# BMI Calculator
This is a simple BMI (Body Mass Index) calculator application, built using **Python, SQLite, Flask, HTML, JavaScript**.

It allows users to input their weight and height to calculate their BMI and determine their weight category.

## Technologies Used
- **Python**: For all backend processing
- **SQLite**: For storing user data and BMI records.
- **Flask**: To create RESTful APIs for handling data operations (create, read, update, delete).
- **HTML**: For structuring the web page.
- **JavaScript**: For handling the BMI calculation logic and user interactions.

## Features
- User-friendly interface for inputting weight and height.
- Real-time BMI calculation and display of results.
- Caregory classification based on BMI values (Underweight, Normal weight, Overweight, Obesity).
- Responsive design for use on various devices.
- Delete BMI records stored in the database.
- Update BMI records stored in the database for existing users.

## How to Use
1. Clone the repository or download the source code.
2. Open the `index.html` file in a web browser. 
3. Enter your weight in kilograms and height in centimeters.
4. Click the "Calculate BMI" button to see your BMI and weight category.
5. Click on "View Records" to list out all existing user data and BMI records stored in the database.
6. To update or delete your BMI records, navigate to the respective buttons in the listed records.
7. Update button prefills the form with existing data of the particular user, allowing you to make changes and save the updated record.
